Calendar Script 

给父类笔记加上 `#calendarRoot`

给笔记添加类似`#Holiday=Christmas`的属性就能添加到日历

类型有这些 
``` 
Event 
Test 
Deadline 
Meeting 
Holiday 
Birthday 
Note 
``` 
可以修改js来扩展
